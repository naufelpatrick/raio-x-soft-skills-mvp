# Aplicações

Exemplos de como o sistema se comporta em comunicação real.

## Princípio

Cada aplicação deve parecer parte do mesmo universo, sem depender de uma composição rígida. Tipografia, contraste, área de respiro, assinatura dourada e clareza da mensagem são os elementos de continuidade.

## Canais

Feed, stories, thumbnails, apresentações, relatórios, newsletters e páginas digitais devem partir da mesma lógica editorial.
