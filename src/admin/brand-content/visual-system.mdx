# Sistema Visual

Um conjunto de regras para que a marca se adapte sem perder reconhecimento.

## Formatos oficiais

Os formatos abaixo são pontos de partida. Cada composição deve preservar hierarquia, margem, respiro e contraste, independentemente do canal.

## Princípios de composição

- Mantenha a hierarquia consistente em todos os formatos.
- Use o ouro sempre como destaque, nunca como fundo extenso.
- Trate o espaço em branco como elemento intencional de design.
- Prefira uma mensagem forte a múltiplos elementos concorrentes.
- Use grid e alinhamento para criar ritmo, não decoração.
